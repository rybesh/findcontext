Find Context Software Trial

Installation Instructions

Thank you for your participation. Instructions for installing the Find Context software are below, please contact kersey@ischool.berkeley.edu if you have any difficulties.

There are three components needed for the Find Context software to work: 1) An up-to-date copy of the Firefox web browser, 2) the Find Context custom Jetpack Add-On, and 3) the Find Context software. We will walk through each process below.

1. Install Firefox

If you don't have the most current version of the Firefox web browser (3.6), Download Firefox and install it according to their directions. Return to this page in your new Firefox browser.

2. Install Jetpack (Find Context version)

Please look over the information in this section first so you know what to expect.

Are you looking at this page in Firefox? Ok, great. If you know that you have installed a previous version of Jetpack, uninstall it now. (This is rare)

When ready, Install our customized Jetpack add-on

  • A warning will pop up when you click the link and you want to press the "allow" button, so that the browser can install it
  • You will then receive a dialog box warning you about malicious software. This is because it is being served from the Berkeley servers instead of Mozilla's. It is safe to disregard this message and click the install button.
  • You will then be prompted to restart Firefox. Please do so.
  • When the software starts up again, if the installation was successful you will see Jetpack welcome screen in a new tab, and this page on a separate tab to the left. You may close the "I am JetPack. Welcome." tab.

3. Install the Find Context Software

The last step! Please read through so you know what to expect, especially since there will be a big anxiety-inducing alert message you will need to ignore.

When ready, Install the Find Context jetpack

  • Please find the username and password you were given in class, you will need it for this step. If you need a new one, contact Noah Kersey at kersey@ischool.berkeley.edu
  • After you click on the install the Find Context jetpack link, you will be prompted to enter your username and password
  • A grey bar will appear along the top of the page if you login successfuly, click the "Install..." button.
  • You will see a big scary red message! This is unavoidable because we are not releasing the software publicly, but we have no interest in your sensitive information. We will not collect your Credit Card numbers or email your grandmother. Because the software is experimental, we cannot avoid the red message, but rest assured, our software will not collect information outside the scope of the study.
  • To continue, click the "I know what I'm doing. Install it!"
  • You will be returned to this page, if the software has installed successfully, you will have a small blue arrow in the top left corner of the browser window. This is the Find Context software.
  • To use the Find Context software, highlight a word in a web page, right click (or control-click on a Mac) and select the 'Query Celtic Studies 138 for "..." to search the reference sources included in this package.' If you have suggestions of other sources which you use and find helpful, please let us know via email.
  • After searching for a word, the menu on the left will expand and you will see the references sources available, those with results are highlighted in green, those without any matching results are highlighted in red. Clicking a reference source will open a new tab with that search.

Thanks for participating in the Find Context software testing. We are very interested to know how it works (or doesn't work) for you. After you have used it once or twice, please complete our survey. We are also interested in hearing about your experience in person, so please consider talking with us in a half-hour interview. Contact Noah Kersey (kersey@ischool.berkeley.edu) to schedule a time.
